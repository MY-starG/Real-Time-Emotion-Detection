FER-2013 Model

FER-2013 is a widely used dataset for facial emotion recognition, featuring 48x48 pixel grayscale images categorized into seven emotions: Angry, Disgust, Fear, Happy, Sad, Surprise, and Neutral. It serves as a benchmark for developing and evaluating emotion recognition algorithms.

My Emotions Model

Emotion Model

My Dataset is a custom facial emotion recognition dataset with 96x96 pixel grayscale images, comprising around 29,000 images categorized into eight emotions: Happy, Sad, Fear, Disgust, Contempt, Surprise, Neutral, and Anger. It follows the same CSV format as FER-2013, containing pixel values for each image.
